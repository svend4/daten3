✨ ОСНОВНЫЕ ХАРАКТЕРИСТИКИ:

Технологии:
  Frontend: React 18, TypeScript, Vite, Tailwind CSS
  Backend: Node.js, Express, TypeScript
  База данных: PostgreSQL (в планах)
  Кэш: Redis (в планах)

Функции:
  ✅ Поиск отелей
  ✅ Поиск авиабилетов
  ✅ Сравнение цен
  ✅ Адаптивный дизайн
  ✅ Modern UI/UX

API Интеграции:
  - Booking.com API (отели)
  - Skyscanner API (авиабилеты)
  - Amadeus API (универсальный)

═══════════════════════════════════════════════════════════════

🚀 БЫСТРЫЙ СТАРТ:

1. Распакуйте архив
2. Frontend: cd frontend && npm install && npm run dev
3. Backend: cd backend && npm install && npm run dev
4. Откройте http://localhost:3001

📖 Подробные инструкции в файле QUICK_START.md
